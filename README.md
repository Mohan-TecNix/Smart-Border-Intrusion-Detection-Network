# Smart Border Intrusion Detection Network

ESP32-based border security system using PIR and vibration sensors for intrusion detection.

## Hardware
- ESP32 DevKit V1
- PIR Motion Sensor
- SW-420 Vibration Sensor
- 16x2 I2C LCD
- Active Buzzer
- LED

## Features
- Motion Detection
- Vibration Detection
- LCD Alerts
- Audio/Visual Alerts
- False Trigger Reduction
